# Render restore notes

If you move the backend off Supabase later:

1. Restore the `backend/` Fastify service.
2. Point `DATABASE_URL` at the target Postgres.
3. Re-enable the Render build/start commands.
4. Set `VITE_API_BASE_URL` back to the Render API host.

Keep this file and `.render.env.example` together as the rollback kit.
